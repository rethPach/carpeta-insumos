# CSV Generator

Proyecto node para generar archivos .cvs para el renderizado en lotes de documentos.

## Instalacion


```bash
1. Git clone
cd carpeta
```

```bash
1. npm install
```

```bash
3. cd src/cerificado-covid-19 
node index.js
```
```bash
4. cd src/resultado-test-covid-19
node index.js
```
